# 🏆 Balonmano Pro

Balonmano Pro es una infografía interactiva desarrollada en HTML, CSS y JavaScript enfocada en enseñar las reglas básicas del balonmano de forma visual, dinámica y atractiva para niños y jóvenes.

La aplicación fue diseñada como una experiencia educativa moderna con enfoque visual, accesibilidad y compatibilidad PWA (Progressive Web App).

---

# ✨ Características

## 🎨 Diseño moderno
- Interfaz futurista inspirada en dashboards deportivos.
- Estilo glassmorphism.
- Animaciones suaves y transiciones interactivas.
- Diseño responsive para computadoras, tablets y celulares.

---

## 🏟️ Infografía interactiva
Incluye:

- Introducción al balonmano.
- Explicación de reglas fundamentales.
- Cancha de balonmano ilustrada correctamente.
- Jugadores posicionados visualmente.
- Elementos gráficos educativos.

---

## 🎮 Mini Quiz Interactivo
Los usuarios pueden responder preguntas educativas sobre:

- Cantidad de pasos permitidos.
- Zona del portero.
- Reglas básicas del juego.

### Características del quiz:
- Botones coloridos y llamativos.
- Feedback instantáneo.
- Diseño amigable para niños.

---

## 📱 PWA (Progressive Web App)

La aplicación puede funcionar como una app instalable.

### Beneficios:
- Acceso rápido desde el celular.
- Compatible offline.
- Experiencia similar a una aplicación móvil.
